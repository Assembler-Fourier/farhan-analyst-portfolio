# Farhan Ahmad — Finance & Accounting Portfolio

Production portfolio for Farhan Ahmad, an MSc International Accounting & Finance graduate based in Dublin and open to opportunities across Ireland.

## Live site

[farhananalyst.com](https://farhananalyst.com)

## Built with

- Next.js App Router, React and TypeScript
- Tailwind CSS plus a custom responsive design system
- React Three Fiber for the interactive finance visual
- Static metadata, structured data, sitemap, robots and social sharing image

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
npm audit --omit=dev
```

## Content and privacy

Professional facts are based on Farhan's CV, final DBS transcript and public LinkedIn profile. The source documents are intentionally not committed, and his full residential address is not published. Market positioning links to original Irish labour-market and finance-industry sources.
